🚀 什么是 Hugo？#

Hugo 是一个基于 Go 语言编写的 静态网站生成器。
它的最大特点是——速度极快、部署方便、几乎零依赖。

使用 Hugo，你可以用 Markdown 写文章，然后自动生成一个完整的博客网站。

🛠️ 安装 Hugo#

在 macOS 上：

brew install hugo
